# Task 1: Data Immersion & Wrangling

## Dataset
Superstore Sales Dataset

## Objective
The objective of this task is to understand the dataset, assess data quality issues, and prepare the dataset for analysis.

## Steps Performed
1. Loaded the dataset using Pandas.
2. Checked dataset structure and shape.
3. Identified missing values.
4. Removed duplicate records.
5. Converted date columns to datetime format.
6. Exported a cleaned dataset.

## Files Included
- Sample - Superstore.csv → Original dataset
- cleaned_superstore.csv → Cleaned dataset
- data_cleaning.py → Python script used for cleaning
- data_dictionary.md → Explanation of dataset columns
